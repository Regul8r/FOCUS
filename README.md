# FOCUS — Ambient Financial Awareness

A physical + digital device that helps college students maintain 
financial awareness without opening a budgeting app.

## How to Run
Terminal 1: python publisher.py
Terminal 2: streamlit run focus_app.py

## Tech Stack
Python · Streamlit · MQTT/HiveMQ · Anthropic Claude API · ElevenLabs

## Setup
Create a .env file with:
ANTHROPIC_API_KEY=your-key
ELEVENLABS_API_KEY=your-key
