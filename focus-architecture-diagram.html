06 — Pseudocode Implementation
Publisher.py — Core Logic DEVICE 1 / Mac Laptop
// ── SETUP ────────────────────────────────── LOAD .env (PLAID_CLIENT_ID, PLAID_SECRET, PLAID_ACCESS_TOKEN) INITIALIZE Plaid client → sandbox environment INITIALIZE MQTT client → connect to HiveMQ broker // ── HEALTH CALCULATION ────────────────────── FUNCTION get_health(account): ratio = account.balance / account.goal IF account.type == "credit": IF ratio <= 0.6: RETURN "green" IF ratio <= 0.9: RETURN "yellow" RETURN "red" ELSE: IF ratio >= 1.0: RETURN "green" IF ratio >= 0.5: RETURN "yellow" RETURN "red" // ── MAIN LOOP ─────────────────────────────── LOOP every 10 seconds: accounts = Plaid.get_balances(ACCESS_TOKEN) FOR each account: account.health = get_health(account) payload = JSON.stringify(accounts) MQTT.publish("focus/accounts", payload) WRITE payload → focus_data.json
focus_pi.py — Core Logic DEVICE 2 / Raspberry Pi 4
// ── SETUP ────────────────────────────────── INITIALIZE NeoPixel rings → GPIO 18, 24 pixels INITIALIZE MQTT client → subscribe focus/accounts INITIALIZE Claude AI client (Anthropic API) INITIALIZE ElevenLabs client INITIALIZE pygame audio → USB speaker hw:3,0 // ── ON MQTT MESSAGE ───────────────────────── ON message received: accounts = JSON.parse(payload) health = get_overall_health(accounts) IF health == "green": ring.fill(GREEN) IF health == "yellow": ring.fill(YELLOW) IF health == "red": ring.fill(RED) SET in_alert_mode = true TIMER(30s) → exit_alert_mode → ambient_breathe // ── SCHEDULED VOICE ───────────────────────── EVERY 30 seconds: read_config() → get user_name, lamp_color, speak_hour IF now.hour == speak_hour AND now.minute == 0: summary = Claude.generate(accounts, user_name, time_of_day) audio = ElevenLabs.speak(summary, voice="Rachel") pygame.play(audio) → USB speaker

FOCUS — Ambient Financial Awareness Device

FOCUS is a physical ambient lamp that connects to your real bank accounts via Plaid and communicates financial health through color and voice. Two NeoPixel rings glow green, yellow, or red based on account health. Every morning, Claude AI generates a personalized summary that ElevenLabs reads aloud by your name.

How It Works

Plaid API pulls live balance data from real accounts (sandbox for demo). Publisher.py calculates health ratios and publishes JSON to HiveMQ via MQTT every 10 seconds. A Raspberry Pi 4 subscribes to that topic, drives the NeoPixel rings, and at the scheduled time calls Claude AI + ElevenLabs to deliver a personalized voice summary through a USB speaker inside the 3D printed lamp.

Running the Prototype

Terminal 1: python3 publisher.py — starts Plaid data flow + MQTT publish

Terminal 2: streamlit run focus_app.py — mobile dashboard on port 8501

Raspberry Pi: sudo python3 focus_pi.py — drives NeoPixels + voice

How Information Moves

  • Plaid API → live bank balances pulled every 10 seconds
  • Publisher.py → calculates green/yellow/red health → MQTT publish to HiveMQ
  • Raspberry Pi → subscribes to MQTT → NeoPixel rings update color
  • Scheduled time → Claude AI generates nudge → ElevenLabs speaks by user name
  • Streamlit dashboard → mobile UI with color picker, time setter, AI insights
